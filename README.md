A study app for the ISACA AAISM (Advanced in AI Security Management) certification exam. The entire app — 300 questions, 120 glossary terms, all logic, 
and all styling — lives in a single downloadable HTML file (≈511 KB) that works on iPhone as a home screen app with no internet required.
